# Data

This public repository uses synthetic data only.

## Folders

| Folder | Purpose |
|---|---|
| `synthetic/` | Generated public-safe demonstration data |
| `external/` | Placeholder for public or synthetic macroeconomic indicators |

## Confidentiality Rule

Do not place real company data in this repository.

The `.gitignore` file blocks common confidential data locations such as:

- `data/raw/`
- `data/private/`
- `data/company/`
