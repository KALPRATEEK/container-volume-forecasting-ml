# Confidentiality Statement

This repository is a public demonstration version of a logistics forecasting project.

It does not contain:

- real company data
- customer data
- internal trade lane information
- internal system exports
- confidential business logic
- proprietary company documentation

All datasets used in this repository are synthetic and created only to demonstrate the technical methodology.

The purpose of this repository is to showcase:

- data analysis
- time-series feature engineering
- machine learning model development
- model evaluation
- dashboarding
- reproducible project structure
- CI/CD automation
